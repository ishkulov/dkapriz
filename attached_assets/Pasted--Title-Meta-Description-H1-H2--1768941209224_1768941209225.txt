# Техническое задание: Оптимизация Title, Meta Description и заголовков H1–H2

## 1. **Цель задачи**

Обеспечить правильную SEO-оптимизацию всех страниц сайта для повышения видимости в поисковых системах (Google и Яндекс), улучшения CTR и релевантности страниц запросам пользователей.

---

## 2. **Объект работы**

* Главная страница сайта
* Категории товаров
* Подкатегории и страницы брендов
* Страницы отдельных товаров
* Статические страницы (например, «О нас», «Доставка», «Контакты»)

---

## 3. **Требования к Title**

1. **Уникальность** для каждой страницы.
2. **Длина**: 50–60 символов (не более 70 символов).
3. **Структура**:

   * Главная страница: «Название сайта – ключевое описание/слоган»
   * Категория: «Название категории – ключевой запрос – бренд/сайт»
   * Товар: «Название товара – категория – бренд»
4. **Ключевые слова**:

   * Основное ключевое слово должно быть в начале.
   * Не использовать «ключевое слово» слишком часто — избегать спама.
5. **Форматирование**: использовать дефис «–» или вертикальную черту «|» для разделения элементов.

---

## 4. **Требования к Meta Description**

1. **Уникальность** для каждой страницы.
2. **Длина**: 150–170 символов (не более 200 символов).
3. **Содержание**:

   * Краткое описание страницы/товара/категории.
   * Включение основных ключевых слов и фраз.
   * Призыв к действию (например, «Купить сейчас», «Посмотреть каталог»).
4. **Читаемость**: описание должно быть информативным и понятным для пользователя, без набора ключевых слов.

---

## 5. **Требования к H1**

1. **Один H1 на странице**.
2. **Уникальность** для каждой страницы.
3. **Содержание**:

   * Главная страница: основной бренд/название сайта.
   * Категории: название категории/темы страницы.
   * Товары: название товара полностью.
4. **Соответствие содержимому** страницы.

---

## 6. **Требования к H2**

1. **Использование для подзаголовков и структурирования контента**.
2. **Количество H2**: несколько на странице, в зависимости от объема контента.
3. **Содержание H2**:

   * Категории: подкатегории, фильтры, уникальные тексты о бренде или типе товара.
   * Товары: характеристики, описание, преимущества.
4. **Включение ключевых слов** без переспама.

---

## 7. **Дополнительные требования**

1. Проверить все страницы на наличие текущих Title, Description, H1–H2.
2. Создать таблицу с предложенными Title, Description и H1–H2 для всех страниц.
3. Настроить Title и Description через CMS (если возможно массовое обновление, использовать массовое редактирование).
4. Проверить корректность отображения тегов через `view-source` и в инструментах типа Google Search Console / Яндекс.Вебмастер.
5. После внедрения провести проверку уникальности через сервисы SEO-анализаторов.

---

## 8. **Ожидаемый результат**

* Все страницы имеют **уникальные и оптимизированные** Title и Meta Description.
* Каждая страница содержит **один H1** и структурированные H2.
* Повышение кликабельности в поисковой выдаче (CTR) и видимости сайта.
